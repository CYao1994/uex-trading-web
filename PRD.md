# UEX Trade Navigator — 产品需求文档 (PRD)

> 产品经理：许清楚 · 产品经理（Product Manager）

## 1. 产品目标

为星际公民（Star Citizen）玩家打造一个**美观、易用、沉浸感强**的 Web 交易路线规划工具，让玩家和朋友能够：

- 快速查询手头货物的最佳清仓路线（最短距离 / 最高利润）
- 查询从某地出发的进货路线
- 分享链接即可使用，无需安装

## 2. 目标用户

- 星际公民玩家（主要：中文社区）
- 有交易需求的新手和老手
- 想要和朋友分享工具的玩家

## 4. 功能范围（MVP = P0 + P1）

### 4.1 核心功能

**A. 库存清仓路线规划（P0）**
- 输入：出发地 + 商品列表（商品名 + SCU 数量）
- 输出：
  - 各商品最高收购价汇总
  - 最短距离路线（最近邻贪心 TSP）
  - 最高利润路线（对比）
  - 最终汇总（收入对比、距离对比）
- 支持：中英文商品名/站名输入

**B. 进货路线查询（P1）**
- 输入：出发地 + 船型 + 本金
- 输出：最优买→卖路线

### 4.2 不在 MVP 范围内
- 用户账号系统
- 路线保存/历史记录
- 实时游戏内数据推送

## 5. 视觉设计要求

### 5.1 星际公民主题元素
- **配色**：深空黑底 + 蓝色/青色科技光效（参考 RSI 官网风格）
- **字体**：科技感无衬线字体
- **动效**：量子跃迁路线动画、星尘粒子背景
- **图标**：飞船剪影、星球、空间站、量子链路
- **装饰**：全息 HUD 风格面板、星图网格线、蓝色发光边框

### 5.2 布局
- 顶部：导航栏（Logo + 功能切换）
- 主区域：左侧输入面板 + 右侧结果展示
- 背景：深空星图 + 粒子动效
- 路线结果：时间线/节点图风格展示各站

### 5.3 交互
- 出发地：搜索下拉框（支持模糊匹配）
- 商品添加：搜索 + 数量输入 + 添加按钮
- 路线切换：最短距离 / 最高利润 Tab 切换
- 加载中：量子跃迁动效

## 6. 技术约束

- UEX API v2.1：GET 端点无需 API Key
- API TLS 兼容性：需要特殊处理（curl --tlsv1.2）
- 前端需代理 API 请求以解决 TLS 和跨域问题
- Python 后端复用现有 sell_inventory.py 核心逻辑
